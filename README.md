# RBFtools

Node for Autodesk Maya to create pose-based output values for driving relationships.

Maya 姿态驱动插件，用于创建基于姿态的输出值来驱动属性关系。

## Installation / 安装

Drag `install.py` into Maya to install, reinstall, or uninstall.

将 `install.py` 拖入 Maya 即可安装、重装或卸载。

## Getting Started / 快速开始

See the in-app help buttons ("?") for detailed usage information on each control.

点击界面上的帮助按钮（"?"）查看每个控件的详细使用说明。
