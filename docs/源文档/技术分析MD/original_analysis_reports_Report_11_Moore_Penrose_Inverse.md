# 第 11 份文件分析报告：Moore-Penrose Inverse

## 1. 数学定义：Penrose 四准则
对于任何矩阵 $\mathbf{A}$，存在唯一的伪逆 $\mathbf{A}^+$。满足弱逆性、最小二乘投影性等。

## 2. 在 RBF 绑定中的价值
* **奇异矩阵挑战:** 当采样姿态非常接近时，常规求逆会导致数值爆炸（Mesh Popping）。
* **最小范数解:** 在欠定系统中，伪逆选出的解具有最小 $L_2$ 范数，对应能量消耗最小化，保证形变平滑。

## 3. 底层实现：基于 SVD
利用 SVD 分解 $\mathbf{A} = \mathbf{U} \Sigma \mathbf{V}^T$。
* **阈值截断 (Thresholding):** 将小于阈值（如 $1e-6$）的奇异值设为 0。这种截断实际上是对系统的“低通滤波”，极大提升了数值稳定性。
