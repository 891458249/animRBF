# 第 13 份文件分析报告：RBF Quaternion Rotation Output

## 1. 核心瓶颈：为什么不能直接线性输出？
四元数分量直接线性插值（LERP）会破坏归一化约束，导致关节塌陷。

## 2. 架构重构：从“直接求解”到“权重分配”
令输出矩阵为单位矩阵 $\mathbf{I}$。RBF 不再直接输出目标值，而是输出一组**样本权重 (Sample Weights)**，描述当前姿态在各采样点中的贡献比例。

## 3. 数学解法：四元数加权平均 (QWA)
1. 构建协方差矩阵 $\mathbf{M} = \sum w_i q_i q_i^T$。
2. 对 $\mathbf{M}$ 进行特征分解。
3. **结论：** 最佳平均四元数是 $\mathbf{M}$ 的**最大特征值所对应的特征向量**。

## 4. 工程意义
该方案具有全局最优性，消除了旋转插值误差导致的蒙皮断裂，是处理肩关节等高自由度关节的完美闭环。
