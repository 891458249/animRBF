# 第 8 份文件分析报告：Chad Vernon - Regularized Linear Regression with RBF

## 1. 系统架构定位
RBF 求解器被定义为多维空间插值系统。本质是一个具备任意输入输出维度的“高级驱动键”。

## 2. 数学基石：正则化线性回归
求解线性方程组 $M \theta = B$。引入岭回归公式：$\theta = (M^T M + \lambda)^{-1} M^T B$。
* **$M$ (特征矩阵):** 输入点之间的距离矩阵。
* **$\lambda$:** 平滑插值结果。

## 3. 工程优化细节
* **特征标准化:** 计算距离前必须标准化，防止不同量级的特征（如 -1000 到 1000 的位移与 0 到 1 的参数）产生偏差。
* **解算器选择:** 建议在 Eigen 中使用 **SVD (奇异值分解)** 计算伪逆，以处理非方阵或奇异矩阵。
* **输入陷阱:** 严禁直接使用多个欧拉角输入，推荐使用四元数或对数映射。
