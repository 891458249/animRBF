# 第 12 份文件分析报告：RBF Quaternion Rotation Input

## 1. 欧拉角的缺陷
直接使用欧拉角输入会导致万向节死锁和多解性跳变 (Flipping)。

## 2. 四元数距离度量
四元数代表球面空间，推荐使用：
$$d(q1, q2) = 1 - |q1 \cdot q2|$$
或者角弧度形式。取绝对值确保了 $q$ 与 $-q$ 代表相同旋转，消除了路径反转问题。

## 3. 混合输入架构
架构上先分别计算标量欧几里得距离和四元数球距，再进行全局重归一化，确保 RBF 半径参数在不同维度下一致。

## 4. 扭转分解 (Swing-Twist Decomposition)
针对四肢关节，将四元数分解为 Swing 和 Twist 通道，防止前臂剧烈扭转时的“糖果纸挤压”效应。
