# 第 9 份文件分析报告：Cholesky Decomposition

## 1. 核心数学定义
将对称正定矩阵 $\mathbf{A}$ 分解为 $\mathbf{A} = \mathbf{L} \mathbf{L}^T$。前提是矩阵必须正定。

## 2. 工程效率：2 倍加速
相比 LU 分解，Cholesky 具有显著优势：
* **计算开销减半:** 复杂度约为 $1/3 n^3$ FLOPs。
* **内存利用率:** 仅访问下三角，提升 Cache 命中率。
* **应用:** 这是《铁拳 8》处理 245 根辅助骨且维持 60FPS 的关键算法。

## 3. 数值稳定性与无平方根变体
工程上常使用 **LDLT 分解**: $\mathbf{A} = \mathbf{LDL}^*$。它避免了开平方根运算，对 GPU 指令集更友好，且在计算时无需行交换。
